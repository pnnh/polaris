# POLARIS — 希波万象资源管理平台

希波万象是一个在线文件资源管理平台，支持多种文件类型，并提供账户认证体系。

## 项目架构

```
┌─────────────────────────────────────────────────────────────┐
│                        浏览器 / 客户端                        │
└──────────────────────────┬──────────────────────────────────┘
                           │
┌──────────────────────────▼──────────────────────────────────┐
│                    polaris（前端）                            │
│              Next.js / React — 端口 7100                     │
└──────────┬────────────────────────────────┬─────────────────┘
           │                                │
┌──────────▼──────────┐        ┌────────────▼────────────────┐
│   portal（Go）       │        │    stargate（.NET 8）        │
│  公共接口 & 认证服务  │        │   需登录的个人/社区/管理接口  │
│  端口 8001           │        │   端口 8101                  │
└──────────┬──────────┘        └────────────┬────────────────┘
           │                                │
┌──────────▼────────────────────────────────▼────────────────┐
│                  PostgreSQL 数据库                           │
│             pagana 提供数据库 DDL 及数据备份                  │
└─────────────────────────────────────────────────────────────┘
```

## 子模块说明

| 模块 | 语言 / 框架 | 职责 |
|---|---|---|
| [polaris](./polaris/README.md) | TypeScript / Next.js 16 | 浏览器端前端网站 |
| [portal](./portal/README.md) | Go / Gin | 公共 API 及账户认证服务 |
| [stargate](./stargate/README.md) | C# / ASP.NET Core (.NET 8) | 需要登录的控制台 API 服务 |
| [pagana](./pagana/README.md) | PostgreSQL | 数据库 DDL 结构及数据备份 |
| [gatekeeper](./gatekeeper/README.md) | TypeScript / Playwright | 端到端（E2E）集成测试 |

## 快速启动

所有服务均提供 Docker 镜像，可通过 `gatekeeper/docker-compose.yaml` 一键启动完整环境：

```bash
cd gatekeeper
docker compose -f docker-compose.yaml up -d
```

> 启动顺序：PostgreSQL → portal → stargate。各服务均配置了健康检查。

## 主要功能

- **账户系统**：注册、登录、登出、会话管理，支持 JWT 鉴权 
- **文件**：文件上传与云端文件管理
- **社区**：社区内容浏览与互动 
- **控制台**：个人控制台与社区管理后台
- **国际化**：前端支持多语言路由
